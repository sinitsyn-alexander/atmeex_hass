# Atmeex Airnanny — Home Assistant Integration

Интеграция для управления бризерами **Atmeex Airnanny** через облачное API `api.iot.atmeex.com`.

## Возможности

- 🌡️ **Климат-контроль** — управление температурой, режимы: выкл / авто / охлаждение
- 🌀 **Вентилятор** — управление скоростью (3 скорости), пресеты: авто / ночной
- 📊 **Датчики** — CO₂ (ppm), температура помещения, влажность, температура приточного воздуха
- 🔌 **Бинарные датчики** — статус подключения (online/offline), тревога отсутствия воды
- 🔘 **Переключатели** — режим охлаждения, ночной режим
- 🔄 **Автообновление данных** каждые 30 секунд
- 🔐 **Два способа авторизации** — по Email+пароль или по телефону+SMS-код

## Установка

### Вариант 1: Ручная установка (для проверки)

1. Скопируйте папку `custom_components/atmeex/` в директорию `/config/custom_components/atmeex/` вашего Home Assistant:

```bash
# Через SSH или Samba
cp -r custom_components/atmeex/ /config/custom_components/atmeex/
```

2. Перезапустите Home Assistant.

3. Перейдите в **Настройки** → **Устройства и службы** → **Добавить интеграцию** → найдите **"Atmeex Airnanny"**.

### Вариант 2: Через HACS

1. В HACS перейдите в **Интеграции** → **⋮** → **Пользовательские репозитории**.
2. Добавьте URL этого репозитория, выберите тип **Integration**.
3. Найдите **"Atmeex Airnanny"** в HACS и установите.
4. Перезапустите Home Assistant.
5. Добавьте интеграцию через **Настройки** → **Устройства и службы**.

## Настройка

1. Выберите способ авторизации:
   - **Email и пароль** — введите email и пароль от аккаунта Atmeex
   - **Телефон и SMS-код** — введите номер телефона, затем SMS-код
2. После успешной авторизации интеграция автоматически обнаружит все ваши бризеры.

## Сущности

На каждое устройство создаются следующие сущности:

| Платформа | Имя | Описание |
|---|---|---|
| Climate | Climate | Управление температурой и режимами |
| Fan | Fan | Управление скоростью вентилятора |
| Sensor | CO2 | Уровень CO₂ в ppm |
| Sensor | Room Temperature | Температура в помещении (°C) |
| Sensor | Humidity | Влажность (%) |
| Sensor | Supply Air Temperature | Температура приточного воздуха (°C) |
| Binary Sensor | Online | Статус подключения |
| Binary Sensor | No Water Alert | Тревога отсутствия воды |
| Switch | Cool Mode | Режим охлаждения |
| Switch | Night Mode | Ночной режим |

## Устранение неполадок

- Если интеграция перестала работать, попробуйте перенастроить её (в меню интеграции → **Перенастроить**).
- Логи доступны в **Настройки** → **Система** → **Журналы** с фильтром `atmeex`.
